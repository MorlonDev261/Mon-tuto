Tutoriel React, Redux & Vite

Guide d'installation et d'utilisation

1. Installer Node.js

Pour installer Node.js dans Termux, exécutez les commandes suivantes :

pkg update && pkg upgrade
pkg install npm
pkg install nodejs-lts
node -v
npm -v

2. Créer une application React avec Vite

Exécutez les commandes suivantes pour configurer une nouvelle application React avec Vite :

npx create-vite@latest my-react-app --template react
cd my-react-app
npm install

3. Installer les dépendances

Installez les dépendances nécessaires :

  • React Router DOM :
    • npm install react-router-dom
  • Material-UI (MUI) :
    • npm install @mui/material @emotion/react @emotion/styled @mui/icons-material
  • Redux Toolkit :
    • npm install @reduxjs/toolkit react-redux

4. Configuration de Redux

Créez un store Redux et configurez-le :

store.js :

import { configureStore, createSlice } from "@reduxjs/toolkit";

const counterSlice = createSlice({
  name: "counter",
  initialState: { value: 0 },
  reducers: {
    increment: (state) => {
      state.value += 1;
    },
    decrement: (state) => {
      state.value -= 1;
    },
  },
});

export const { increment, decrement } = counterSlice.actions;

const store = configureStore({
  reducer: {
    counter: counterSlice.reducer,
  },
});

export default store;

6. Démarrer le serveur de développement

Exécutez la commande suivante pour démarrer le serveur :

npm run dev

Votre application sera disponible à l'adresse suivante : http://localhost:5173.
